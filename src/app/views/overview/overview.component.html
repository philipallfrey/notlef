This site is an entry by Philip Allfrey for the 2020 ACDH-CH Virtual Hackathon. The task was to design and implement a creative visualisation of the data and metadata contained in the Digital Humanities Course Registry.

Click on the thumbnails on this page or the arrows in the header to navigate to the different visualisations of the data.

On each page one set of data is highlighted in colour. Clicking on one of these values will filter the other data by that page (e.g. to show information for a particular language or country).

Each set of information has an icon beside it. If the information is taken directly from the Digitial Humanities Course Registry API, the is displayed, and clicking on it will show you the endpoint and parameters that were used to query the API. If I have done any significant processing of the data, the is displayed, and clicking on it will allow you to download that data in CSV format.
